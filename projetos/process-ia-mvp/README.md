# Process IA MVP

Sistema web para mapeamento de processos de negócio com foco em:

- levantamento do processo atual
- análise As Is
- sugestões iniciais de To Be
- apoio a Lean, Six Sigma e BPM

## Tecnologias

### Backend
- FastAPI
- SQLAlchemy
- SQLite

### Frontend
- React
- Vite
- Axios
- React Router

## Funcionalidades do MVP

- cadastro de processos
- cadastro de etapas do fluxo
- análise automática do As Is
- identificação de gargalos
- identificação de desperdícios Lean
- recomendações iniciais de To Be
