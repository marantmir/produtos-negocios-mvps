# Process IA Platform

Plataforma para mapeamento de processos, análise As Is, sugestões To Be e apoio a Lean, Six Sigma, BPMN e Pesquisa Operacional.

## Módulos previstos
- autenticação
- processos e etapas
- dashboard
- BPMN visual
- coleta assistida por IA
- relatórios PDF
- deploy em cloud gratuita
