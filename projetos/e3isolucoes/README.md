# E3I Soluções — pacote pronto para Netlify

## Estrutura
- `index.html`: página pronta para publicar
- `assets/logo-e3i.png`: logo quadrado
- `netlify.toml`: configuração básica

## Publicação rápida no Netlify
1. Acesse o painel do Netlify.
2. Escolha **Add new site** > **Deploy manually**.
3. Arraste a pasta `e3i-netlify` ou o arquivo `.zip`.
4. Após publicar, edite o número de WhatsApp em `index.html` se necessário e republique.

## Ajustes recomendados antes de publicar
- Substituir `5516999999999` pelo número real da empresa.
- Atualizar título, descrição e textos finais com dados comerciais reais.
- Conectar domínio próprio, se desejar.
